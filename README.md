# Eco-Drones | Smart Eco-Friendly Drone

## Project Overview
Eco-Drones is a **fictional eco-friendly drone** product designed for photography, package delivery, and sustainable flight using solar power.  
This project is a **responsive single-page website** created using **HTML5 and CSS3 only** with a fully functional pre-order form.

---

## Seed / Design Inspiration
- Seed: Eco-friendly drones for everyday use.
- Focus: Clean, modern, and green-themed UI.
- Goal: Demonstrate responsive layout, smart form, accessibility, and CSS interactivity.

---

## Features
1. **Responsive Layout** using CSS Grid + Flexbox  
2. **Three CSS-only interactive components**:
   - Hover effect on feature cards  
   - Navigation link hover highlights  
   - Button hover with subtle animation  
3. **Smart Form**:
   - Text inputs: Name & Email  
   - Checkboxes: Interests selection  
   - Radio buttons: Plan selection  
   - Dropdown: Favorite color  
   - Textarea: Additional notes  
   - Submit button with CSS styling  
4. **Accessibility**:
   - Proper labels for inputs  
   - Focus states for keyboard navigation
